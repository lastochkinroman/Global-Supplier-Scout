# Market Research Analyzer Bot

> Telegram-бот, который находит лучших поставщиков и считает реальную цену закупки

Напишите список товаров — бот найдёт поставщиков, посчитает все затраты (доставка, хранение, таможня) и выдаст Excel-отчёт с рекомендациями от AI.

## Что умеет

- Ищет поставщиков из базы 10+ международных компаний
- Считает полную стоимость с доставкой и всеми сборами
- Анализирует риски через Groq AI
- Генерирует готовые Excel-отчёты
- Работает быстро

## Установка

**1. Поставьте библиотеки:**
```bash
pip install -r requirements.txt
```

**2. Создайте `.env`:**
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
GROQ_API_KEY=ваш_ключ_groq
```

Токены:
- Telegram — [@BotFather](https://t.me/botfather)
- Groq — [console.groq.com](https://console.groq.com)

**3. Запустите:**
```bash
python main.py
```

## Как пользоваться

Напишите боту список товаров через запятую:
```
wireless earbuds, smart watch, yoga mat
```

Получите Excel-файл с:
- Ценами от разных поставщиков
- Полной стоимостью (товар + доставка + хранение + таможня)
- Рейтингами и сроками доставки
- Рекомендациями AI, у кого выгоднее брать

## Что в базе

- **Электроника**: наушники, умные часы, колонки, powerbank
- **Спорт**: коврики для йоги, фитнес-трекеры, бутылки
- **Дом и офис**: лампы, зарядки, органайзеры
- **Аксессуары**: рюкзаки, чехлы, очки
